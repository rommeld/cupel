# Contributing to Cupel

At the moment I build `cupel` alone, with help from Anthropic's Claude and Moonshot's Kimi. I welcome contributions from anyone, but please understand that I won't be able to implement everything.

## Reporting bugs and requesting features

Please open a GitHub issue first.

- **Feature request:** a short description or summary of what you'd like to see.
- **Bug report:** a short description of the bug, the expected behavior, and the actual behavior.

If you already have an idea for the feature implementation or the fix, feel free to open a PR directly instead.

## Submitting a PR

If your PR was created with the help of an coding agent, please include the prompt(s) you used in the PR description. This makes it much easier for me to follow along and understand the resulting code.
